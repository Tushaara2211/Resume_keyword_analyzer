# Resume Keyword Analyzer
A Python tool to analyze and optimize your resume with job description keywords for ATS.

## Features
- Keyword Extraction
- Resume Analysis
- CSV Report
- PDF Support

## Requirements
- PyPDF2
- nltk
- pandas
Install: `pip install -r requirements.txt`
